# Vidtube

Vidtube is a YouTube-inspired backend application built using the MERN ecosystem technologies. It provides the core backend functionality required for a video sharing platform, including user authentication, video management, comments, and channel interactions. The project focuses on building scalable RESTful APIs and efficient data handling for modern web applications.

## Features

- User Authentication and Authorization (JWT based)
- Video Upload and Management
- Comment System for Videos
- Channel Creation and Interaction
- RESTful API Architecture
- Secure Data Handling
- MongoDB Database Integration
- API Testing with Postman

## Tech Stack

- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JSON Web Tokens (JWT)  
- **Testing:** Postman
